# Discord Bot Toolkit
I'm building a discord toolkit for making discord bots in an easier and simpler way than the discord python module offers.
This is one of those weekend projects that I build for personal use mostly but I'm sharing this as it might come out useful if you're making discord bots and want a cleaner approach.
